# KeenKeeper

## 📌 Project Description
KeenKeeper is a personal relationship management web application that helps users maintain meaningful connections. It allows users to track interactions, monitor relationship status, and receive reminders to stay in touch with friends, family, and colleagues.

---

## 🛠 Technologies Used
- React (Frontend framework)
- Vite (Build tool)
- Tailwind CSS (Styling)
- React Router DOM (Navigation)
- Recharts (Data visualization)
- React Toastify (Notifications)

---

## ⭐ Key Features

### 1. Friend Management Dashboard
Users can view and manage all their connections in one place, including status indicators such as On-Track, Almost Due, and Overdue.

### 2. Timeline Tracking
The timeline page records all interactions (calls, texts, meetups, video calls), helping users track their communication history.

### 3. Friendship Analytics
Provides visual insights into interaction types using charts, helping users understand their communication patterns.

## ⚙️ Why React and Vite?

This project uses React as the main frontend framework because it allows building reusable components and managing the user interface efficiently. It helps keep the code organized and makes the application easier to maintain and scale.

Vite is used as the build tool because it provides a fast development experience. It starts the development server quickly and updates changes instantly using hot module replacement. This makes development smoother and saves time compared to traditional tools.
